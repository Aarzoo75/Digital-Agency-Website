### Run Locally

To run **Pixology** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/pixology.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/pixology.git
```
